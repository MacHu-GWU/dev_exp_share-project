Z-Order
==============================================================================


Overview
------------------------------------------------------------------------------
Z-Order 最早是 1966 提出的一项将多维数据映射到一维的方法. 随着数据库技术的发展, 这种映射方法由于其特性, 被应用到了数据库技术中, 特别是在大数据时代再次被提及, 在 Delta Lake, Hudi, Iceberg 中都有应用. 本文是我在研读博文 "深入分析 Z-Order" 时自己的理解和总结.
